MovingBox is designed with an intuitive navigation system to help you manage your inventory efficiently. This guide will help you understand the app's layout and how to navigate between different sections.

## Main Navigation

MovingBox is organized into main sections accessible through intuitive navigation links:

### Dashboard

<InfoBlock>
The Dashboard is your central hub that provides an at-a-glance view of your inventory.
</InfoBlock>

From your Dashboard, you can:
- View a summary of your complete inventory
- See your total estimated inventory value
- Access inventory statistics organized by location
- Access inventory statistics organized by label


### Locations

<InfoBlock>
The Locations tab organizes your items based on where they are stored.
</InfoBlock>

In this section, you can:
- Browse locations (rooms, storage units, etc.)
- View all items within a specific location
- Add new locations or edit existing ones
- See the number of items and total value per location

### Item Management

<InfoBlock>
The "+" button opens the item adding interface, where enhanced AI-powered features help document your belongings with unprecedented detail.
</InfoBlock>

When adding items in MovingBox 2.0, you can:
- Take multiple photos from different angles for comprehensive documentation
- Review AI-generated details with enhanced accuracy and detail recognition
- Access high-resolution AI analysis (MovingBox Pro)
- Manually add items with 15+ new properties
- Assign locations, categories, and moving priorities
- Attach documents like receipts and warranties

### Item List

<InfoBlock>
The Item List provides a complete view of your entire inventory with powerful new bulk operation capabilities.
</InfoBlock>

Features in the Item List include:
- Browse all items in a sortable list with multi-select checkboxes
- Enhanced search for specific items using keywords
- Advanced filtering by category, location, value range, and new properties
- Sort items by name, date added, estimated value, or any custom field
- Perform bulk operations on multiple selected items
- Export selected items to various formats

## Navigation Gestures

MovingBox uses standard iOS gestures for intuitive navigation:

| Gesture | Action |
|---------|--------|
| Swipe left/right | Navigate between items in detail view |
| Swipe down | Dismiss a sheet and return to previous view |

## Common Tasks

### Finding Items Quickly

1. Tap the search icon on the Items or Locations screen
2. Enter keywords related to the item you're looking for

### Organizing Your Inventory

Effective organization makes managing your inventory easier:

- **Labels**: Group similar items (Electronics, Furniture, etc.)
- **Locations**: Track where items are stored (Living Room, Storage, etc.)

### Viewing Item Details

To see complete information about any item:

1. Tap the item from any list view
2. View or edit detailed information
