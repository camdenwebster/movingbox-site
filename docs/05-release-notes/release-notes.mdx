# 2.1.0
**Released:** December 31, 2025

## New Features

### Multi-Item Detection
Photograph multiple items at once and let AI identify each one separately. Perfect for cataloging shelves, cabinets, or entire rooms quickly.

### Redesigned Import & Export
- **Import:** Preview data before importing, real-time progress tracking, clear error messages
- **Export:** Visual progress feedback, streamlined sharing, better reliability for large inventories

### iPad & Mac Navigation
Split-view interface with sidebar navigation and multi-column layouts optimized for larger screens.

### Pre-Populated Rooms
New users start with common rooms (Kitchen, Living Room, Bedroom, Bathroom) including SF Symbol icons.

### Enhanced Sorting
Sort items by name, date, location, or value with persistent preferences.

### Improved Onboarding
Optional survey to understand how you plan to use MovingBox, helping us prioritize the right features.

### Feature Discovery
"What's New" screen highlights new features after updates (Settings → About → What's New).

## Bug Fixes & Improvements

- Fixed crash when deleting multiple items
- Improved thumbnail loading and memory management
- Faster import processing and optimized image handling
- Better overall responsiveness
- Added thoughtful App Store review prompts
